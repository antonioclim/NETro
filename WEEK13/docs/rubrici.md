# Rubrici de Evaluare — Săptămâna 13

## Criterii Laborator (15p din nota finală)

### Scanare Porturi (3p)
- 3p: Toate porturile identificate corect, output formatat
- 2p: Porturi identificate, output minimal
- 1p: Parțial funcțional
- 0p: Nu funcționează

### MQTT Client (3p)
- 3p: Pub/sub funcțional, statistici calculate
- 2p: Comunicare de bază funcțională
- 1p: Conectare reușită, comunicare parțială
- 0p: Nu se conectează

### Captură Trafic (3p)
- 3p: Captură completă, analiză corectă
- 2p: Captură salvată, analiză superficială
- 1p: Doar captură, fără analiză
- 0p: Lipsă captură

### Documentare (3p)
- 3p: Raport complet cu toate secțiunile
- 2p: Raport parțial
- 1p: Doar răspunsuri la întrebări
- 0p: Nedocumentat

### Participare (3p)
- 3p: Activ, întrebări relevante
- 2p: Prezent, participare moderată
- 1p: Prezent, participare minimă
- 0p: Absent

## Criterii Proiect Echipă — Security Audit (10p)

- Lista porturi expuse (2p)
- Verificare credențiale (2p)
- Criptare comunicații (2p)
- Logging configurat (2p)
- Plan remediere (2p)
